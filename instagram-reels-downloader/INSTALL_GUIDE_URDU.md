# Instagram Reels Downloader - Installation Guide (Urdu/Hindi)

## Chrome Extension Kaise Install Karein

### Step 1: Files Download Karein
Sab se pehle, ensure karein ke aap ke paas ye saari files hain:
```
instagram-reels-downloader/
├── manifest.json
├── popup.html
├── popup.js
├── content.js
├── background.js
├── lib/
│   └── jszip.min.js
├── icons/
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md
```

### Step 2: Chrome Mein Extension Load Karein

1. **Chrome Browser Kholein**
   
2. **Extensions Page Par Jayein**
   - Address bar mein type karein: `chrome://extensions/`
   - Ya phir: Settings → Extensions

3. **Developer Mode Enable Karein**
   - Top right corner mein "Developer mode" ka toggle hai
   - Usay ON karein

4. **Extension Load Karein**
   - "Load unpacked" button par click karein
   - `instagram-reels-downloader` folder select karein
   - OK karein

5. **Extension Install Ho Gaya!**
   - Ab aap extension ko Chrome toolbar mein dekh sakte hain

## Istemal Ka Tareeqa (Usage)

### Reels Download Karna:

1. **Instagram Profile Par Jayein**
   - Kisi bhi public Instagram profile ki reels page par jayein
   - Example: `https://www.instagram.com/username/`

2. **Extension Icon Par Click Karein**
   - Chrome toolbar mein extension ka icon hoga
   - Us par click karein

3. **"Download All Reels" Button Dabayein**
   - Popup mein button dikhai dega
   - Us par click karein

4. **Intezaar Karein**
   - Extension automatically:
     - Saari reels scroll karega
     - Sab reels collect karega
     - Ek ZIP file banayega
     - Download shuru ho jayega

5. **Downloads Folder Check Karein**
   - Aap ke downloads folder mein ZIP file aa jayegi
   - Usay extract karein aur reels dekhein

## Zaroori Baatein (Important Notes)

### ✅ Ye Kaam Karega:
- Public Instagram profiles
- Jin profiles mein reels hain
- Stable internet connection

### ❌ Ye Kaam Nahi Karega:
- Private profiles
- Blocked accounts
- Bina internet ke

### ⚠️ Ehtiyaat (Precautions):
1. Sirf public profiles use karein
2. Copyright ka khayal rakhein
3. Zyada tez download na karein (Instagram block kar sakta hai)
4. Instagram ki Terms of Service follow karein

## Masail Ka Hal (Troubleshooting)

### Extension Nahi Chal Raha?
- Page refresh karein (F5)
- Chrome restart karein
- Check karein ke extension enabled hai

### Reels Download Nahi Ho Rahin?
- Profile public hai ya private check karein
- Internet connection check karein
- Kuch der intezaar kar ke dobara try karein

### ZIP File Nahi Ban Rahi?
- Downloads folder mein jagah check karein
- Browser console mein errors dekhein (F12 → Console)
